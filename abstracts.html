INFORMS-APS 2019, Brisbane Australia

The 20th INFORMS Applied Probability Society Conference, July 3-5, 2019, Brisbane Australia.


Abstract submission


To submit an abstract please do the following:

  1. Submit your proposal for a talk via this form.
  2. Use this LaTeX template to create your abstract. Instructions are in the template.(*)
  3. Send your latex template to informs-aps@uq.edu.au with title: "Talk Abstract".

(*) -- The file has been updated on February 5, 2019 and it now includes a link to the .sty file if a speaker wants to compile it directly on their computer.
